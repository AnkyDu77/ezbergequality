Hi there! This is a REST API for traders. Here you can register yourself as a trader, create your strategy, place/cancel your orders and get info about them.
Notice, all parameters are required to be filled in.

WARNING: This API is just a prototipe! Application is still in development mode! STABILITY NOT GUARANTEED!!! Use it at your own risk!
WARNING: Format "number.decimals" is required everywhere "float" pointer is set. Example: "10000.00", not "10000"


FUNCTIONS LIST:

1. Trader's Registration: '/traderRegistration/'
Parameters:
- cryptowallet (str);
- username (str)

Request format:
/traderRegistration/cryptowallet=[yourCryptoAddress]&username=[yourUsername]


2. Strategy Registration: '/strategyRegistration/'
Parameters:
- cryptowallet (str);
- algorithm openness - open or close - (str);
- strategy's max capital capacity - small (<= $100'000), medium (<= $1'000'000), big (<= $100'000'000), infinite (<= 1'000'000'000) - (str);
- leverage (int);
- strategy name

Request format:
/strategyRegistration/cryptowallet=[yourCryptoAddress]&algoOpenness=[yourAlgoOpenness]&capitalCapacity=[yourCapitalCapacity]&leverage=[yourLeverage]&strategyName=[yourStrategyName]


3. Place Order: '/placeOrder/'
Parameters:
- cryptowallet (str);
- strategyID (str);
- symbol (str);
- direction (str);
- order type - have to be "LIMIT" for now - (str);
- price (float);
- volume (float);
- time in force - have to be "GTC" for now (str).

Request format:
'/placeOrder/cryptowallet=[yourCryptoAddress]&strategyID=[yourStrategyID]&symbol=[symbol]&direction=[yourOrdersDirection]&orderType=[yourOrderType]&price=[yourOrdersPrice]&volume=[yourOrdersVolume]&timeInForce=[yourOrdersTimeInForce]'


4. Get All Open Orders: '/openOrders/'
Parameters:
- cryptowallet (str);
- strategyID (str);

Request format:
/openOrders/cryptowallet=[yourCryptoAddress]&strategyID=[yourStrategyID]


5. Get All Orders: '/getOrders/'
Parameters:
- cryptowallet (str);
- strategyID (str);

Request format:
/getOrders/cryptowallet=[yourCryptoAddress]&strategyID=[yourStrategyID]


6. Cancel particular order: /cancelParticularOrder/
Parameters:
- cryptowallet (str);
- strategyID (str);
- orderID (str);

Request format:
/cancelParticularOrder/cryptowallet=[yourCryptoAddress]&strategyID=[yourStrategyID]&orderID=[yourOrderID]


7. Cancel All Orders: '/cancelOpenOrders/'
Parameters:
- cryptowallet (str);
- strategyID (str);

Request format:
/getOrders/cryptowallet=[yourCryptoAddress]&strategyID=[yourStrategyID]
